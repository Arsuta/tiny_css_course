Урок 6.7 - Стилизация профильной карточки

1. CSS Custom Properties и их применение

CSS Custom Properties (пользовательские свойства) - это мощный механизм для создания переиспользуемых значений в CSS. Они особенно полезны при работе с темами и компонентами.

/* Определение переменных в корневом элементе */
:root {
    --primary: #466A85;         /* Основной цвет */
    --secondary: #717777;       /* Вторичный цвет */
    --imageSize: 200px;         /* Размер изображения */
    --spacing-base: 8px;        /* Базовый отступ */
}

Преимущества использования CSS переменных:

  • Централизованное управление значениями
  • Возможность изменения значений через JavaScript
  • Поддержка каскадности и области видимости
  • Упрощение поддержки темной темы
/* Пример использования calc() с переменными */
.card {
    padding: calc(var(--spacing-base) * 2);
    margin-top: calc(var(--imageSize) / 3);
}

2. Глубокое понимание Flexbox

Flexbox работает с двумя осями: главной (main axis) и поперечной (cross axis). Направление главной оси определяется свойством flex-direction.

/* Основные свойства Flexbox */
.flex-container {
    display: flex;              /* Активация flex-контекста */
    flex-direction: row;        /* Направление главной оси */
    justify-content: space-between; /* Распределение по главной оси */
    align-items: center;        /* Выравнивание по поперечной оси */
    gap: 10px;                 /* Отступы между элементами */
}

.flex-item {
    flex-basis: 200px;         /* Базовый размер элемента */
    flex-grow: 1;              /* Коэффициент роста */
    flex-shrink: 0;            /* Коэффициент сжатия */
}

5. Ключевые моменты урока

  • CSS переменные делают код более поддерживаемым и гибким
  • Flexbox предоставляет мощные инструменты для выравнивания
  • object-fit необходим для правильной работы с изображениями
  • Свойство gap упрощает создание отступов между элементами
  • Позиционирование элементов может комбинировать различные подходы
