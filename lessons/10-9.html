Урок 10.9 - Стили при наведении и фокусе

Разница между :focus и :focus-visible

Псевдоклассы :focus и :focus-visible имеют разное поведение:

Особенности:

  • :focus - срабатывает при любом фокусе
  • :focus-visible - только при навигации с клавиатуры
  • Доступность - :focus-visible улучшает UX
  • Поддержка - :focus-visible поддерживается не везде

:focus

:focus-visible

/* Стили для focus и focus-visible */
.form-control:focus {
    border-color: #466A85;
    box-shadow: 0 0 0 3px rgba(70, 106, 133, 0.2);
}

.form-control:focus-visible {
    outline: 2px solid #466A85;
    outline-offset: 1px;
}

Стили при наведении

Hover эффекты помогают пользователю понять интерактивность элементов:



/* Стили для hover */
.form-control:hover {
    border-color: #bbb;
    background-color: #fafafa;
}

.btn:hover {
    background: #385674;
    transform: translateY(-2px);
}

Комбинация состояний

Можно комбинировать разные состояния для создания сложных эффектов:

/* Комбинированные состояния */
.form-control:hover:focus {
    border-color: #385674;
    box-shadow: 0 0 0 4px rgba(70, 106, 133, 0.3);
}

.btn:hover:focus {
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);
}
