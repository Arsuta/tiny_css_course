Урок 3.3: Применение стилей к SVG

Теория

Способы стилизации SVG

SVG элементы можно стилизовать несколькими способами. Рассмотрим основные подходы и их особенности.

Важно помнить

SVG использует специальные атрибуты для стилизации, которые отличаются от обычных CSS свойств:

  • fill вместо background-color
  • stroke вместо border-color
  • stroke-width вместо border-width

Пример стилизации через CSS

.svg-element {
    fill: #466A85;        /* Цвет заливки */
    stroke: #2c4a5e;      /* Цвет обводки */
    stroke-width: 2;      /* Толщина обводки */
    opacity: 0.8;         /* Прозрачность */
}

Наследование стилей

SVG элементы могут наследовать некоторые стили от родительских элементов:

.svg-container {
    color: blue;          /* Базовый цвет, наследуется как fill */
    font-size: 16px;      /* Размер шрифта для text элементов */
}

.svg-container svg {
    fill: currentColor;   /* Использование текущего цвета из color */
}

Анимация SVG стилей

.animated-svg {
    fill: #3498db;                /* Начальный цвет */
    transition: fill 0.3s ease;   /* Плавное изменение цвета */
}

.animated-svg:hover {
    fill: #e74c3c;               /* Цвет при наведении */
}
