Урок 11.5 - Стилизация элементов

Кнопка поделиться

Кнопка является основным элементом взаимодействия:

/* Стили для кнопки */
.share-button {
    width: 48px;
    height: 48px;
    border-radius: 50%;
    border: none;
    background: #466A85;
    color: white;
    cursor: pointer;
    transition: all 0.3s ease;
    display: flex;
    align-items: center;
    justify-content: center;
}
