Урок 12.8 - CSS функции

Что такое CSS функции?

CSS функции позволяют выполнять вычисления и манипуляции с значениями в стилях. Они упрощают работу с цветами, размерами и другими параметрами, делая код более динамичным и адаптивным.

Зачем использовать CSS функции?

  • Упрощение вычислений - позволяет выполнять математические операции прямо в CSS.
  • Управление переменными - использование переменных для упрощения изменений стилей.
  • Адаптивность - возможность задавать значения, которые автоматически подстраиваются под размеры экрана.

Основные CSS функции:

  • calc() - позволяет выполнять математические вычисления для задания значений.
  • var() - используется для работы с CSS переменными.
  • clamp() - позволяет задавать значения с ограничениями, что полезно для адаптивного дизайна.
  • min() и max() - выбирают минимальное или максимальное значение из списка, что позволяет управлять размерами элементов.

Использование calc()

Функция calc() позволяет выполнять математические операции:

С помощью calc() можно комбинировать разные единицы измерения, например, проценты и пиксели.

/* Пример использования calc() */
.element {
    width: calc(100% - 20px);
}
Пример с использованием calc()

Применение calc():

С помощью calc() можно задавать размеры, которые зависят от других значений, что делает макеты более гибкими.

Использование var()

Функция var() позволяет использовать CSS переменные:

CSS переменные упрощают управление стилями, так как их значения можно изменять в одном месте, и они будут применяться ко всем элементам, использующим эту переменную.

/* Определение переменных */
:root {
    --main-color: #466A85;
}
.element {
    background-color: var(--main-color);
}

Применение var():

С помощью var() можно легко изменять темы и стили, просто меняя значения переменных.

Использование clamp()

Функция clamp() позволяет задавать значения с ограничениями:

Это полезно для адаптивного дизайна, так как позволяет задавать минимальные и максимальные размеры для элементов.

/* Пример использования clamp() */
.element {
    font-size: clamp(1rem, 2vw + 1rem, 2rem);
}

Применение clamp():

С помощью clamp() можно задавать размеры, которые будут адаптироваться к размеру экрана, оставаясь в заданных пределах.
