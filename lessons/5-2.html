Урок 5.2: Вёрстка на Grid

Теория

Создание базовой сетки

Для создания сетки карточек мы будем использовать CSS Grid Layout. Начнем с базовой структуры:

.cards-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

Свойство grid-template-columns: repeat(3, 1fr) создает три равные колонки.

Адаптивная сетка

Для создания адаптивной сетки используем auto-fit и minmax:

.responsive-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 20px;
}

Пояснение свойств:

  • auto-fit - автоматически определяет количество колонок
  • minmax(300px, 1fr) - задает минимальную и максимальную ширину колонки
  • gap - устанавливает отступы между элементами сетки

Медиа-запросы

Для адаптации сетки под разные устройства используем медиа-запросы:

/* Планшеты */
@media (max-width: 768px) {
    .cards-grid {
        grid-template-columns: repeat(2, 1fr);
    }
}

/* Мобильные устройства */
@media (max-width: 480px) {
    .cards-grid {
        grid-template-columns: 1fr;
    }
}

Практика

Задание: Создание адаптивной сетки карточек

Создайте адаптивную сетку карточек, которая будет автоматически перестраиваться под разные размеры экрана.

Требования:

  • Используйте CSS Grid для создания сетки
  • Карточки должны быть шириной минимум 300px
  • Добавьте отступы между карточками 20px
  • Сделайте адаптивную версию с помощью медиа-запросов

Подсказки:

  • Используйте display: grid для создания сетки
  • Примените repeat(auto-fit, minmax(300px, 1fr)) для адаптивных колонок
  • Добавьте gap для отступов между карточками
  • Не забудьте про box-shadow и border-radius для карточек
