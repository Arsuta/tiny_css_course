Урок 10.1 - Введение в стилизацию форм

Введение

В этой главе мы изучим различные техники стилизации форм и их элементов:

Основные темы главы:

  • Стилизация текстовых полей и email-инпутов
  • Оформление select и textarea
  • Кастомизация радиокнопок и чекбоксов
  • Работа с выпадающими меню
  • Стилизация меток и легенд
  • Обработка состояний hover и focus
/* Базовые стили для форм */
.form-control {
    width: 100%;
    padding: 8px 12px;
    border: 2px solid #ddd;
    border-radius: 4px;
    font-size: 16px;
    transition: all 0.3s ease;
}

.form-control:focus {
    border-color: #466A85;
    outline: none;
    box-shadow: 0 0 0 3px rgba(70, 106, 133, 0.2);
}
