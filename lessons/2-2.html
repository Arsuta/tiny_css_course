Урок 2.2: Display grid

Теория

Свойство display: grid

Свойство display: grid создает контекст сетки, превращая элемент в grid-контейнер. Это базовое свойство для начала работы с CSS Grid Layout.

display: grid

Создает блочный grid-контейнер

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

display: inline-grid

Создает строчный grid-контейнер

.container {
    display: inline-grid;
    grid-template-columns: repeat(2, 100px);

Вложенные сетки

Создание вложенных сеток

.outer-grid {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
    gap: 20px;
}

.inner-grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

Важные особенности

  • Grid-контейнер создает новый контекст форматирования
  • Прямые потомки становятся grid-элементами автоматически
  • Можно создавать неограниченное количество вложенных сеток
  • Сетки могут комбинироваться с Flexbox
