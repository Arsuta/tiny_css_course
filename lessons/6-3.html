Урок 6.3: Создание фона

Теория

Радиальный градиент

Функция radial-gradient() создает изображение с плавным переходом между цветами, расходящимися из центральной точки.

/* Базовый синтаксис */
background: radial-gradient(shape size at position, color1, color2);

/* Пример */
background: radial-gradient(circle at center, #466A85, #7be6e8);
Базовый радиальный градиент

Позиционирование градиента

Можно указать позицию центра градиента с помощью ключевых слов или значений:

/* Позиционирование в правом верхнем углу */
background: radial-gradient(circle at top right, #466A85, #7be6e8);

/* Позиционирование с процентами */
background: radial-gradient(circle at 75% 25%, #466A85, #7be6e8);
Позиционированный градиент

Размер градиента

Размер градиента можно контролировать с помощью ключевых слов:

/* Ключевые слова размера */
closest-corner  /* До ближайшего угла */
farthest-corner /* До самого дальнего угла */
closest-side    /* До ближайшей стороны */
farthest-side   /* До самой дальней стороны */

/* Пример */
background: radial-gradient(circle closest-corner at center, #466A85, #7be6e8);
Градиент с указанием размера

Применение в нашем проекте

Для карточки профиля мы создадим фон с радиальным градиентом в верхней части:

.card-header {
    height: 120px;
    background: radial-gradient(
        circle at top right,
        var(--primary-color),
        var(--secondary-color)
    );
}

Практика

Задание: Создание градиентного фона для карточки

Создайте радиальный градиент для верхней части карточки профиля.

Требования:

  • Используйте radial-gradient для создания фона
  • Настройте позицию центра градиента
  • Используйте CSS переменные для цветов
  • Добавьте правильные размеры для header

Подсказки:

  • Используйте height для задания высоты header
  • Примените radial-gradient() для создания градиента
  • Укажите позицию с помощью at position
  • Используйте переменные var(--color-name)
