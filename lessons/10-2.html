Урок 10.2 - Сброс стилей fieldset

Проблемы стандартных стилей fieldset

Элемент fieldset имеет встроенные стили браузера, которые часто требуют модификации для современного дизайна. Рассмотрим основные проблемы и их решения:

Стандартные стили браузера:

  • Рамка: 2px groove (трехмерная)
  • Отступы: 0.35em 0.75em 0.625em
  • Минимальная ширина: min-inline-size: min-content
  • Отступ для legend: Специальное позиционирование

Стилизация legend

Элемент legend имеет особое поведение и требует специального подхода к стилизации:

Стандартный legend

Заголовок группы

Стилизованный legend

Заголовок группы 
/* Расширенная стилизация legend */
legend {
    float: left;           /* Убираем странные отступы */
    padding: 0;
    margin: 0 0 1rem;
    width: 100%;           /* Растягиваем на всю ширину */
    font-weight: bold;
    color: #466A85;
}

/* Очистка для float */
fieldset {
    display: flow-root;    /* Создаем новый контекст форматирования */
}

Особенности legend:

  • Позиционирование: Legend всегда отображается поверх верхней границы fieldset
  • Размер: Может влиять на минимальную ширину fieldset
  • Отступы: Имеет специфичные правила для отступов
  • Выравнивание: Может требовать особых приемов для центрирования
