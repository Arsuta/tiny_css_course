Урок 8.4 - Средние экраны

1. Выравнивание чисел по правому краю

На средних экранах важно правильно выравнивать числовые значения для лучшей читаемости. Рассмотрим основные принципы:

/* Выравнивание чисел */
.number-cell {
    text-align: right;    /* Выравнивание по правому краю */
    font-family: monospace;    /* Моноширинный шрифт для чисел */
    padding-right: 20px;    /* Отступ справа для читаемости */
}

/* Медиа-запрос для средних экранов */
@media (min-width: 768px) {
    .number-cell {
        font-size: 1.1em;    /* Увеличенный размер для средних экранов */
    }
}

Почему это важно?

  • Читаемость: Выравнивание по правому краю облегчает сравнение чисел
  • Моноширинный шрифт: Все цифры имеют одинаковую ширину
  • Отступы: Создают визуальное разделение между колонками

2. Выравнивание колонок

Для средних экранов важно оптимальное расположение колонок. Рассмотрим варианты компоновки:

/* Адаптивная сетка колонок */
.columns-layout {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 20px;
}

/* Медиа-запрос для средних экранов */
@media (min-width: 768px) {
    .columns-layout {
        grid-template-columns: repeat(2, 1fr);
    }
}

Первая колонка

Содержимое первой колонки с выравниванием по левому краю.

Вторая колонка

Содержимое второй колонки с числовыми значениями справа.

Преимущества адаптивной сетки:

  • Гибкость: Автоматическая адаптация под ширину экрана
  • Отзывчивость: Плавное изменение размеров
  • Оптимизация: Эффективное использование пространства

3. Поля ввода с выравниванием

Числовые поля ввода требуют особого внимания к выравниванию и форматированию:

/* Стили для числовых полей */
.number-input {
    text-align: right;
    font-family: monospace;
    padding-right: 10px;
}

/* Медиа-запрос для средних экранов */
@media (min-width: 768px) {
    .input-group {
        display: grid;
        grid-template-columns: 150px 1fr;
    }
}

Особенности числовых полей:

  • Выравнивание: Правое выравнивание для чисел
  • Форматирование: Единый формат для валюты
  • Доступность: Понятные метки и подсказки

4. Отступы и поля для средних экранов

Правильное использование отступов и полей критично для средних экранов:

/* Отступы для разных размеров экрана */
@media (min-width: 768px) {
    .content-block {
        padding: 20px;
        margin-bottom: 30px;
    }

    .content-block.comfortable {
        padding: 30px;
        margin-bottom: 40px;
    }
}

Пример с отступами для средних экранов

Принципы использования отступов:

  • Иерархия: Отступы создают визуальную иерархию
  • Читаемость: Правильные отступы улучшают восприятие
  • Адаптивность: Отступы меняются в зависимости от экрана
