План работы

Мы разобьем создание проекта на несколько этапов:

  1. Настройка CSS custom properties для управления цветами и размерами
  2. Создание фона с помощью radial-gradient
  3. Центрирование карточки с помощью flexbox
  4. Позиционирование и стилизация изображения профиля
  5. Настройка размера и позиции фоновых элементов
  6. Стилизация текстового контента
  7. Добавление и стилизация кнопок действий

Базовая структура

Начнем с создания базовой HTML структуры для нашей карточки:

/* Базовая структура HTML */
<div class="profile-card">
    <div class="card-header"></div>
    <img src="profile.jpg" alt="Profile" class="profile-image">
    <div class="card-content">
        <h2>Имя пользователя</h2>
        <p>Должность</p>
    </div>
</div>

В следующих уроках мы будем постепенно добавлять стили и функциональность к этой базовой структуре.

Практика

Задание: Создание базовой структуры карточки профиля

Создайте базовую HTML структуру для карточки профиля, которую мы будем улучшать в следующих уроках.

Требования:

  • Создайте основной контейнер для карточки
  • Добавьте секцию для фонового изображения
  • Разместите изображение профиля
  • Создайте секцию для контента (имя, должность)

Подсказки:

  • Используйте семантические теги для структуры
  • Добавьте классы для будущей стилизации
  • Подготовьте изображение профиля
  • Продумайте структуру контента
