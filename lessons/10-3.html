Урок 10.3 - Стилизация полей ввода

Текстовые поля и email

Текстовые поля являются основой большинства форм. При их стилизации важно учитывать несколько аспектов:

Ключевые моменты:

  • Размер шрифта: Минимум 16px для мобильных устройств
  • Отступы: Достаточные для удобного клика
  • Состояния: Normal, Focus, Hover, Disabled
  • Доступность: Контрастные цвета и чёткие границы
/* Расширенные стили для текстовых полей */
input[type="text"],
input[type="email"],
input[type="password"] {
    line-height: 1.5;
    color: #333;
    background: #fff;
}

/* Состояния */
.form-control:hover {
    border-color: #bbb;
}

.form-control:disabled {
    background: #f5f5f5;
    cursor: not-allowed;
}

Select и Textarea

Select и textarea имеют свои особенности стилизации, которые нужно учитывать:

Особенности select:

  • appearance: none - убирает нативный вид
  • Кастомная стрелка - через background-image
  • Отступ справа - для стрелки
/* Стилизация select */
select.form-control {
    appearance: none;
    background-image: url('data:image/svg+xml...');
    background-repeat: no-repeat;
    background-position: right 12px center;
    padding-right: 36px;
}

/* Стилизация textarea */
textarea.form-control {
    min-height: 100px;
    resize: vertical;
    line-height: 1.5;
}

Radio и Checkbox

Для создания кастомных radio и checkbox используется техника скрытия оригинального input и создания собственной разметки:

Принцип работы:

  • Скрытие оригинала: opacity: 0 вместо display: none
  • Кастомная разметка: Через псевдоэлементы
  • Состояния: Использование :checked
  • Доступность: Сохранение фокуса и клавиатурной навигации
/* Базовая структура */
.custom-radio,
.custom-checkbox {
    position: relative;
    padding-left: 30px;
    cursor: pointer;
}

/* Скрытие оригинального input */
.custom-radio input,
.custom-checkbox input {
    position: absolute;
    opacity: 0;
}

/* Стили для отмеченного состояния */
.custom-radio input:checked ~ .radio-mark::after {
    content: '';
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    width: 10px;
    height: 10px;
    background: white;
    border-radius: 50%;
}

Общие рекомендации

При стилизации форм важно помнить:

  • Доступность: Сохраняйте возможность навигации с клавиатуры
  • Состояния: Обязательно стилизуйте все состояния элементов
  • Отзывчивость: Учитывайте мобильные устройства
  • Валидация: Предусмотрите стили для ошибок и успешных состояний
  • Консистентность: Поддерживайте единый стиль всех элементов формы
