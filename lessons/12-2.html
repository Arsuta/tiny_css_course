Урок 12.2 - CSS Grid

Что такое CSS Grid?

CSS Grid Layout - это мощный инструмент для создания сложных макетов на веб-страницах. Он позволяет организовывать элементы в виде сетки, что упрощает процесс верстки и делает его более управляемым.

Основные концепции CSS Grid:

  • Сетка - состоит из строк и столбцов, которые формируют ячейки.
  • Ячейки - области, в которые помещаются элементы.
  • Области - именованные группы ячеек, которые можно использовать для упрощения разметки.

Преимущества CSS Grid:

  • Двухмерная система - возможность работать как с рядами, так и со столбцами
  • Гибкость - легко адаптировать макеты под разные размеры экранов
  • Упрощение кода - меньше CSS-кода для сложных макетов
  • Легкость в управлении - простое изменение порядка элементов

Создание сетки

Для создания сетки необходимо использовать свойство display: grid:

/* Пример сетки */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

.grid-item {
    background: #f7f5e8;
    padding: 20px;
    border-radius: 8px;
}

Свойства CSS Grid

Основные свойства, которые используются для настройки сетки:

Ключевые свойства:

  • grid-template-columns - определяет количество и ширину столбцов
  • grid-template-rows - определяет количество и высоту рядов
  • grid-gap - задает расстояние между ячейками сетки
  • grid-area - позволяет задавать область для элемента
/* Пример использования свойств */
.grid-container {
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: auto;
    grid-gap: 10px;
}

.grid-item {
    grid-area: 1 / 1 / 2 / 2; /* Задает область для элемента */
}
