Урок 1.4: Добавление CSS в HTML

Теория

Способы добавления CSS в HTML

1. Внешний CSS файл

<!-- В head HTML документа -->
<link rel="stylesheet" href="styles.css">

/* styles.css */
body {
    margin: 0;
    font-family: Arial, sans-serif;
}
✅ Преимущества
  • Лучшая производительность (кэширование)
  • Чистое разделение контента и стилей
  • Один файл для множества страниц
  • Меньший размер HTML
  • Удобство поддержки
⚠️ Особенности
  • Дополнительный HTTP-запрос
  • Необходимость организации файловой структуры

2. Внутренние стили (style тег)

<head>
    <style>
        body {
            margin: 0;
            font-family: Arial, sans-serif;
        }
    </style>
</head>
✅ Преимущества
  • Нет дополнительных HTTP-запросов
  • Удобно для одностраничных сайтов
  • Быстрое прототипирование
❌ Недостатки
  • Увеличивает размер HTML
  • Нельзя переиспользовать стили
  • Сложнее поддерживать

3. Инлайн стили (атрибут style)

<div style="color: blue; margin: 20px;">
    Текст с инлайн-стилями
</div>
⚠️ Не рекомендуется использовать, потому что:
  • Нарушает принцип разделения задач
  • Высокая специфичность затрудняет переопределение
  • Невозможно переиспользовать стили
  • Сложно поддерживать
  • Увеличивает размер HTML

Порядок загрузки CSS

1. User Agent Stylesheets

Встроенные стили браузера (можно сбросить через reset.css)

2. Внешние стили

CSS файлы, подключенные через link

3. Внутренние стили

Стили в теге style

4. Инлайн стили

Стили в атрибуте style

Атрибуты для link

rel="stylesheet"

Обязательный атрибут, указывающий тип связи

<link rel="stylesheet" href="styles.css">

media

Определяет, когда применять стили

<link rel="stylesheet" 
    href="print.css" 
    media="print">

type="text/css"

Необязателен в HTML5

<!-- Устаревший вариант -->
<link rel="stylesheet" 
    type="text/css" 
    href="styles.css">

Лучшие практики

Организация CSS файлов

styles/
├── main.css      # Основные стили
├── reset.css     # Сброс стилей
├── variables.css # Переменные
└── components/   # Компоненты
    ├── header.css
    ├── footer.css
    └── buttons.css

Порядок подключения

<head>
    <link rel="stylesheet" href="reset.css">
    <link rel="stylesheet" href="variables.css">
    <link rel="stylesheet" href="main.css">
    <link rel="stylesheet" href="components.css">
</head>

Оптимизация производительности

  • Минифицируйте CSS файлы
  • Используйте CDN для популярных библиотек
  • Добавляйте media атрибуты где нужно
  • Объединяйте мелкие CSS файлы

Важно помнить

  • Порядок подключения CSS важен - последующие стили могут переопределять предыдущие
  • Используйте внешние CSS файлы как основной метод добавления стилей
  • Инлайн стили следует использовать только в крайних случаях
  • Правильная организация CSS файлов упрощает поддержку проекта

Практика

Задание

Попрактикуйтесь в использовании всех трёх способов добавления CSS. Создайте страницу с тремя параграфами и примените к каждому стили разными способами.
