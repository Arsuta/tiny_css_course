Урок 9.7 - Тени

1. Box-shadow vs Drop-shadow

В CSS есть два основных способа создания теней: box-shadow и filter: drop-shadow. Давайте детально рассмотрим каждый из них:

Сравнение свойств:

  • box-shadow - создаёт прямоугольную тень вокруг блока элемента, включая padding и border
  • drop-shadow - создаёт тень, точно повторяющую форму контента, включая прозрачные области
  • Производительность - box-shadow требует меньше вычислительных ресурсов
  • Поддержка - box-shadow поддерживается всеми современными браузерами
  • Множественные тени - box-shadow позволяет создавать несколько теней через запятую
  • Внутренние тени - только box-shadow может создавать внутренние тени (inset)

Параметры box-shadow:

/* Синтаксис box-shadow */
box-shadow: h-offset v-offset blur spread color inset;

/* Примеры */
box-shadow: 0 4px 8px rgba(0,0,0,0.1);                    /* Простая тень */
box-shadow: inset 0 0 10px rgba(0,0,0,0.2);              /* Внутренняя тень */
box-shadow: 0 4px 8px #000, inset 0 0 4px #000;          /* Комбинация теней */

Настройка параметров тени:

box-shadow
drop-shadow
inner shadow
multiple shadows

2. Текстовые тени

Текстовые тени помогают улучшить читаемость и создать визуальные эффекты. Рассмотрим различные варианты применения:

Применение text-shadow:

  • Улучшение контраста - тонкая тень для лучшей читаемости на сложном фоне
  • Декоративные эффекты - множественные тени для создания объёма
  • Неоновый эффект - яркие размытые тени для имитации свечения
  • 3D эффект - направленные тени для создания объёма
/* Текстовые тени */
/* Базовая тень */
.card-number {
    text-shadow: 1px 1px 2px rgba(0,0,0,0.3);
}

/* Неоновый эффект */
.neon-text {
    text-shadow: 0 0 5px #fff,
                 0 0 10px #fff,
                 0 0 20px #0ff,
                 0 0 30px #0ff;
}

/* 3D эффект */
.text-3d {
    text-shadow: 1px 1px 0 #ccc,
                 2px 2px 0 #999,
                 3px 3px 0 #666;
}
