Урок 10.7 - Стилизация кнопки отправки

Базовая стилизация кнопок

Кнопка отправки формы должна быть заметной и понятной для пользователя:

Важные аспекты:

  • Размер - достаточный для удобного клика
  • Цвет - контрастный и привлекающий внимание
  • Состояния - hover, active, disabled
  • Отзывчивость - визуальная обратная связь
/* Базовые стили кнопки */
.btn {
    display: inline-block;
    padding: 12px 24px;
    border: none;
    border-radius: 4px;
    background: #466A85;
    color: white;
    cursor: pointer;
    transition: all 0.3s ease;
}

Состояния кнопки

Разные состояния кнопки помогают пользователю понять, что происходит:

Состояние загрузки

Неактивное состояние

/* Состояния кнопки */
.btn:hover {
    background: #385674;
    transform: translateY(-2px);
}

.btn:disabled {
    opacity: 0.6;
    cursor: not-allowed;
}

Варианты дизайна

Разные варианты оформления для разных ситуаций:

/* Варианты кнопок */
.btn-primary {
    background: linear-gradient(45deg, #466A85, #7be6e8);
}

.btn-outline {
    background: transparent;
    border: 2px solid #466A85;
    color: #466A85;
}
