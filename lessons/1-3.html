Урок 1.3: Что такое CSS

Теория

CSS: История и эволюция

1994: Зарождение CSS

Хокон Виум Ли предложил CSS как решение проблемы стилизации веб-страниц, когда HTML становился перегруженным презентационными элементами.

1996: CSS1

Первая спецификация CSS включала базовые возможности: шрифты, цвета, отступы, границы.

1998: CSS2

Добавлены позиционирование, z-index, медиа-типы и поддержка различных устройств.

2011: CSS3

Модульный подход, анимации, градиенты, flexbox и многое другое.

Основные концепции CSS

Селекторы

Определяют, к каким элементам применяются стили

/* Селектор элемента */
p { color: blue; }

/* Селектор класса */
.highlight { background: yellow; }

/* Селектор ID */
#header { font-size: 24px; }

Свойства

Определяют, какие аспекты элемента изменяются

/* Типографика */
font-size: 16px;
font-weight: bold;

/* Цвета и фоны */
color: #333;
background-color: white;

/* Размеры и отступы */
width: 100%;
padding: 20px;

Значения

Определяют конкретные параметры свойств

/* Абсолютные единицы */
width: 100px;

/* Относительные единицы */
font-size: 1.5em;

/* Проценты */
width: 50%;

/* Ключевые слова */
display: flex;

Способы подключения CSS

1. Внешний файл CSS

<link rel="stylesheet" href="styles.css">
Преимущества:
  • Разделение контента и стилей
  • Кэширование браузером
  • Легкое обслуживание

2. Внутренние стили

<style>
    body { background: #fff; }
</style>
Недостатки:
  • Смешивание HTML и CSS
  • Нет кэширования
  • Сложно поддерживать

3. Инлайн стили

<div style="color: blue;"></div>
Не рекомендуется:
  • Нарушает SoC
  • Сложно поддерживать
  • Высокая специфичность

Базовый синтаксис CSS

selector {
    property: value;
    /* Комментарий */
    property2: value2;
}
  • selector - определяет целевые элементы
  • property - что именно мы хотим изменить
  • value - новое значение свойства
  • ; - разделитель между декларациями
  • {} - обрамляют блок деклараций

Важные концепции

  • CSS работает каскадно - стили могут наследоваться и перезаписываться
  • Специфичность определяет приоритет правил
  • CSS является декларативным языком
  • Современный CSS поддерживает вычисления и переменные

Практика

Задание

Создайте несколько CSS правил для стилизации текста. Попрактикуйтесь в написании правильного синтаксиса CSS.
