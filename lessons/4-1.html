Работа со шрифтами

Шрифты являются одним из ключевых элементов веб-дизайна. Правильный выбор и настройка шрифтов может значительно улучшить читаемость и внешний вид вашего сайта.

Комбинации шрифтов

Montserrat + Open Sans

Пример комбинации шрифтов, где заголовки используют Montserrat, а основной текст - Open Sans. Это популярное сочетание для современных веб-сайтов.

/* Подключение шрифтов */
@import url('https://fonts.googleapis.com/css2?family=Montserrat:wght@300;400;700&family=Open+Sans:wght@300;400;700&display=swap');

/* Стили для заголовков */
h1, h2, h3, h4, h5 {
    font-family: 'Montserrat', sans-serif;
}

/* Стили для основного текста */
body {
    font-family: 'Open Sans', sans-serif;
    font-size: 16px;
    line-height: 1.6;
}

/* Различные размеры */
.small-text { font-size: 12px; }
.medium-text { font-size: 16px; }
.large-text { font-size: 20px; }

Свойство font-weight

Свойство font-weight определяет насколько жирным или тонким должен быть шрифт. Оно может принимать числовые значения от 100 до 900 или ключевые слова.

Текст с изменяемой жирностью

/* Числовые значения */
.thin { font-weight: 100; }
.regular { font-weight: 400; }
.bold { font-weight: 700; }
.black { font-weight: 900; }

/* Ключевые слова */
.normal { font-weight: normal; }  /* 400 */
.bold { font-weight: bold; }      /* 700 */

    line-height: 1.6;  /* Стандартный */

/* Межбуквенные интервалы */

    letter-spacing: -0.5px;  /* Плотный */

    letter-spacing: 1px;     /* Разреженный */
}
