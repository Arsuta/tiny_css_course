Урок 7.2 - Стилизация цитаты

1. Работа со шрифтами для цитат

Единицы измерения в CSS

При работе с текстом в CSS используются различные единицы измерения. Рассмотрим самые популярные:

Абсолютные единицы:

/* Абсолютные единицы измерения */
h1 {
    font-size: 16px;   /* Пиксели - самая базовая единица */
    font-size: 12pt;   /* Пункты - 1pt = 1/72 дюйма */
    font-size: 1cm;    /* Сантиметры */
    font-size: 1mm;    /* Миллиметры */
    font-size: 1in;    /* Дюймы - 1in = 96px */
}

Относительные единицы:

/* Относительные единицы измерения */
p {
    font-size: 1em;    /* Относительно размера шрифта родителя */
    font-size: 1rem;   /* Относительно размера шрифта корневого элемента */
    font-size: 100%;   /* Процент от размера шрифта родителя */
    font-size: 1vw;    /* 1% от ширины viewport */
    font-size: 1vh;    /* 1% от высоты viewport */
    font-size: 1ch;    /* Ширина символа "0" текущего шрифта */
}

Демонстрация различных единиц измерения:

Текст размером 1.5rem

Текст размером 150%

Когда какие единицы использовать:

  • px - когда нужен точный размер, независимый от настроек пользователя
  • em - для элементов, размер которых должен зависеть от родителя
  • rem - для создания масштабируемой типографики, зависящей от корневого размера
  • % - для адаптивного дизайна и масштабирования относительно родителя
  • vw/vh - для размеров, зависящих от размера окна браузера

Интерактивная демонстрация:

Текст 1.5em (относительный)

Текст 1.5rem (относительный корню)

Текст 150% (относительный)

Для создания выразительных цитат мы будем использовать декоративный шрифт Passions Conflict. Этот шрифт особенно хорошо подходит для коротких текстов и заголовков.

/* Импорт шрифта */
@import url('https://fonts.googleapis.com/css2?family=Passions+Conflict&display=swap');

/* Применение шрифта к цитате */
blockquote {
    font-family: 'Passions Conflict', cursive;
    font-size: 2em;
    line-height: 1;
}

4. Важные моменты

  • Декоративные шрифты лучше использовать для коротких текстов
  • Line-height для декоративных шрифтов часто требует корректировки
  • Важно учитывать читаемость при выборе размера шрифта
  • Цитаты могут быть оформлены как с помощью тега blockquote, так и с помощью других элементов
