Урок 10.4 - Стилизация выпадающих меню

Нативный select

Рассмотрим базовую стилизацию нативного select элемента:

Особенности нативного select:

  • Ограниченная стилизация - не все свойства работают
  • Разный вид - в разных браузерах и ОС
  • Доступность - встроенная поддержка клавиатуры
/* Стилизация select */
.custom-select {
    appearance: none;
    padding: 10px 35px 10px 12px;
    border: 2px solid #ddd;
    border-radius: 4px;
    background: #fff;
    cursor: pointer;
}

/* Кастомная стрелка */
.select-arrow {
    position: absolute;
    pointer-events: none;
}
