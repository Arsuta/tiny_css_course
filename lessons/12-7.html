Урок 12.7 - Псевдоклассы

Что такое псевдоклассы?

Псевдоклассы позволяют применять стили к элементам в зависимости от их состояния или положения в документе. Они начинаются с одиночного двоеточия (:).

Основные псевдоклассы:

  • :hover - применяется при наведении курсора на элемент.
  • :focus - применяется, когда элемент получает фокус (например, при нажатии на него).
  • :active - применяется в момент нажатия на элемент.
  • :nth-child(n) - применяется к элементам в зависимости от их позиции в родительском элементе.

Пример использования псевдоклассов

Рассмотрим пример, как использовать псевдоклассы для стилизации элементов:

Наведите курсор на меня! 
/* Пример использования псевдоклассов */
.pseudo-class:hover {
    background: #466A85;
    color: white;
}

Использование :nth-child

Псевдокласс :nth-child позволяет стилизовать элементы в зависимости от их позиции в родительском элементе. Это очень полезно для создания различных эффектов и стилей для четных и нечетных элементов списка.

Синтаксис :nth-child

Синтаксис псевдокласса выглядит следующим образом:

:nth-child(an + b)
  • a - шаг, с которым выбираются элементы (например, 2 для каждого второго элемента).
  • b - смещение, которое определяет, с какого элемента начинать выбор (например, 1 для первого элемента).

Примеры использования :nth-child

  • li:nth-child(odd) - выбирает все нечетные элементы списка.
  • li:nth-child(even) - выбирает все четные элементы списка.
  • li:nth-child(3) - выбирает третий элемент списка.
  • li:nth-child(2n) - выбирает каждый второй элемент, начиная со второго.
  • li:nth-child(2n + 1) - выбирает каждый нечетный элемент, начиная с первого.
/* Пример использования :nth-child */
li:nth-child(odd) {
    background: #f0f0f0;
}

li:nth-child(even) {
    background: #d0d0d0;
}
