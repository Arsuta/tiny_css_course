Урок 3.4: Анимация элементов в CSS

Теория

Ключевые кадры (Keyframes)

Ключевые кадры позволяют определить состояния анимации в разные моменты времени. Это основа любой CSS-анимации.

@keyframes moveLeftRight {
    0% { transform: translateX(-50px); }
    100% { transform: translateX(50px); }
}

.box {
    animation: moveLeftRight 1s infinite alternate;
}

Длительность анимации (Duration)

Свойство animation-duration определяет время выполнения одного полного цикла анимации. Это ключевой параметр, влияющий на скорость анимации.

Основные характеристики:

  • Значение задаётся в секундах (s) или миллисекундах (ms)
  • Нельзя использовать отрицательные значения
  • Значение 0s отключает анимацию
  • Можно указывать дробные значения (например, 0.5s)
/* Базовый синтаксис */
.element {
    animation-name: myAnimation;     /* Имя анимации */
    animation-duration: 2s;          /* Длительность */
}

/* Примеры различных значений */
.quick {
    animation-duration: 0.5s;        /* Быстрая анимация */
}
.normal {
    animation-duration: 2s;          /* Средняя скорость */
}
.slow {
    animation-duration: 4s;          /* Медленная анимация */
}
.milliseconds {
    animation-duration: 500ms;       /* Значение в миллисекундах */
}

Рекомендации по использованию:

  • Для интерфейсных анимаций используйте короткие длительности (0.2s - 0.5s)
  • Для декоративных анимаций можно использовать более длительные значения
  • Слишком долгие анимации могут раздражать пользователей
  • Учитывайте контекст использования при выборе длительности

Количество повторений (Iteration Count)

Свойство animation-iteration-count определяет, сколько раз должна повториться анимация. Это позволяет контролировать цикличность анимации.

Возможные значения:

  • Числовое значение (целое или дробное)
  • infinite (бесконечное повторение)
  • По умолчанию: 1 (одно выполнение)
/* Различные способы использования */
.once {
    animation-iteration-count: 1;      /* Одно выполнение */
}
.three-times {
    animation-iteration-count: 3;      /* Три повторения */
}
.infinite {
    animation-iteration-count: infinite; /* Бесконечное повторение */
}
.partial {
    animation-iteration-count: 2.5;    /* Два с половиной повторений */
}

/* Пример с полным объявлением анимации */
.element {
    animation-name: rotate;            /* Имя анимации */
    animation-duration: 2s;            /* Длительность */
    animation-iteration-count: 3;      /* Количество повторений */
    animation-timing-function: linear; /* Функция времени */
}

Важные замечания:

  • Дробные значения приводят к остановке анимации в промежуточном состоянии
  • Значение infinite следует использовать с осторожностью
  • Для интерфейсных элементов лучше использовать конечное число повторений
  • Можно комбинировать с другими свойствами анимации
