Урок 12.6 - Псевдоэлементы

Что такое псевдоэлементы?

Псевдоэлементы позволяют стилизовать определенные части элемента, не добавляя дополнительных HTML-тегов. Они начинаются с двойного двоеточия (::) и могут использоваться для создания различных эффектов.

Основные псевдоэлементы:

  • ::before - добавляет контент перед содержимым элемента.
  • ::after - добавляет контент после содержимого элемента.
  • ::first-line - стилизует первую строку текста элемента.
  • ::first-letter - стилизует первую букву текста элемента.

Пример использования псевдоэлементов

Рассмотрим пример, как использовать псевдоэлементы для добавления контента:

Пример с псевдоэлементами 
/* Пример использования псевдоэлементов */
.pseudo-element::before {
    content: ;
    position: absolute;
    top: -10px;
    left: 10px;
    background: #466A85;
    color: white;
    padding: 5px;
    border-radius: 4px;
}

.pseudo-element::after {
    content: ;
    position: absolute;
    bottom: -10px;
    right: 10px;
    background: #466A85;
    color: white;
    padding: 5px;
    border-radius: 4px;
}
