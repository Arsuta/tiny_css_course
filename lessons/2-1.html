Урок 2.1: CSS Grid

Теория

Что такое CSS Grid?

CSS Grid Layout - это мощная система макетов, которая позволяет создавать двумерные сетки для веб-страниц. В отличие от Flexbox, который работает в одном измерении, Grid позволяет управлять как строками, так и столбцами одновременно.

Основные термины

Grid Container

Элемент, к которому применяется display: grid

Grid Items

Прямые потомки grid-контейнера

Grid Lines

Линии, формирующие структуру сетки

Grid Tracks

Пространство между двумя соседними линиями

Базовый пример Grid

.container {
    display: grid;
    grid-template-columns: 200px 1fr 200px;
    grid-template-rows: auto 1fr auto;
    gap: 20px;
}

Единицы измерения в Grid

fr (fraction)

Гибкая единица, распределяющая доступное пространство

grid-template-columns: 1fr 2fr 1fr;
/* Пропорции: 25% 50% 25% */

auto

Размер определяется содержимым

grid-template-columns: auto 1fr auto;

minmax()

Задает минимальное и максимальное значения

grid-template-columns: minmax(200px, 1fr);

Важно помнить

  • Grid - это двумерная система макетов
  • Можно комбинировать фиксированные и гибкие размеры
  • fr единицы распределяют оставшееся пространство
  • gap создает отступы между элементами, но не по краям

Fraction Units (fr)

Fr единица представляет собой долю свободного пространства в сетке.

.fr-demo {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;
    /* 1fr = 25%, 2fr = 50%, 1fr = 25% */
}
