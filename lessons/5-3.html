Использование background-clip

Свойство background-clip позволяет создать эффект градиентного текста:

.gradient-title {
    background: linear-gradient(45deg, #466A85, #7be6e8);
    -webkit-background-clip: text;
    background-clip: text;
    color: transparent;
}

Градиентный текст

Обратите внимание, как градиент применяется только к тексту

Важные моменты:

  • Свойство background-clip: text обрезает фон по контуру текста
  • Необходимо использовать префикс -webkit- для поддержки в разных браузерах
  • Цвет текста должен быть transparent, чтобы был виден градиент

Практика

Задание: Создание градиентного заголовка

Создайте заголовок с градиентным текстом, используя свойство background-clip.

Требования:

  • Используйте шрифт Arial или другой sans-serif
  • Размер текста должен быть не менее 32px
  • Примените градиент к тексту с помощью background-clip
  • Добавьте префикс -webkit- для кроссбраузерности

Подсказки:

  • Используйте font-size и font-weight для настройки шрифта
  • Создайте градиент с помощью linear-gradient()
  • Примените background-clip: text и -webkit-background-clip: text
  • Установите color: transparent для отображения градиента
