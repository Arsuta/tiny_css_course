Урок 10.5 - Стилизация меток и легенд

Стилизация меток (labels)

Метки играют важную роль в формах, обеспечивая понятность и доступность:

Основные принципы:

  • Размер шрифта: Должен быть читаемым
  • Отступы: Обеспечивают визуальную связь с полем
  • Цвет: Достаточный контраст для читаемости
  • Обязательные поля: Четкое обозначение
Минимум 3 символа
/* Стили для меток */
.form-label {
    display: block;
    margin-bottom: 8px;
    color: #466A85;
    font-weight: bold;
}

/* Обязательные поля */
.required::after {
    content: '*';
    color: #e74c3c;
}

Стилизация легенд (legends)

Легенды используются для группировки связанных полей:

Контактная информация 
/* Стили для легенд */
legend {
    padding: 0 10px;
    font-weight: bold;
    color: #466A85;
}

fieldset {
    border: 2px solid #ddd;
    border-radius: 8px;
    padding: 20px;
}

Инлайновые метки

Для радиокнопок и чекбоксов часто используются инлайновые метки:

Выберите опции 
/* Стили для инлайновых меток */
.inline-label {
    display: inline-flex;
    align-items: center;
    margin-right: 20px;
}
