Урок 10.6 - Стилизация placeholder

Основы стилизации placeholder

Placeholder - это подсказка внутри поля ввода, которая исчезает при вводе текста:

Особенности placeholder:

  • Браузерные префиксы - для полной поддержки
  • Ограниченные свойства - не все CSS свойства работают
  • Доступность - не должен заменять label
  • Контраст - должен быть достаточным для читаемости
/* Базовая стилизация placeholder */
.form-control::placeholder {
    color: #999;
    font-style: italic;
    opacity: 0.7;
}

Анимация placeholder

Можно добавить интерактивность при фокусе на поле:

/* Анимация placeholder */
.placeholder-fade:focus::placeholder {
    opacity: 0;
    transition: opacity 0.3s ease;
}

.placeholder-slide:focus::placeholder {
    transform: translateY(-100%);
    opacity: 0;
    transition: all 0.3s ease;
}

Браузерные префиксы

Для максимальной поддержки используются префиксы:

/* Префиксы для разных браузеров */
::-webkit-input-placeholder { /* Chrome/Safari/Opera */
    color: #999;
}
::-moz-placeholder { /* Firefox */
    color: #999;
}
:-ms-input-placeholder { /* IE/Edge */
    color: #999;
}
