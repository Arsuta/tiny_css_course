Урок 4.5: Финальные штрихи

Теория

Выравнивание и переносы текста

Для создания профессионально выглядящего текста важно правильно настроить его выравнивание и переносы.

По левому краю

Этот текст выровнен по левому краю. Это стандартное выравнивание для большинства текстов.

По центру

Этот текст выровнен по центру. Часто используется для заголовков.

По правому краю

Этот текст выровнен по правому краю. Редко используется для основного текста.

По ширине

Этот текст выровнен по ширине с автоматическими переносами слов. Создает ровные края текстового блока.

Это пример текста с выравниванием по ширине (justify) и автоматическими переносами слов. Длинные слова будут автоматически переноситься, чтобы сохранить ровные края текста. Это особенно важно при создании газетного макета.

/* Настройка выравнивания и переносов */
.text {
    text-align: justify;
    hyphens: auto;
    -webkit-hyphens: auto;
    -ms-hyphens: auto;
}

Свойство hyphens: auto включает автоматические переносы слов, но требует указания языка в HTML (lang="ru").

Обтекание изображений текстом

Свойство float позволяет создать эффект обтекания изображения текстом, что часто используется в газетных макетах.

Это пример текста, который обтекает изображение. Изображение расположено слева, а текст обтекает его справа. Такой приём часто используется в газетных статьях для создания более динамичного и интересного макета.

При использовании float важно помнить о clearfix для родительского элемента, чтобы избежать проблем с высотой контейнера.

/* Настройка обтекания */
img {
    float: left;
    margin: 0 20px 20px 0;
}

/* Clearfix для родительского элемента */
.wrap-demo {
    overflow: hidden;
}

Использование max-width и margin: auto

Для создания отзывчивого макета с ограниченной шириной используется комбинация max-width и margin: auto.

Этот блок имеет максимальную ширину 800px и автоматически центрируется на странице. При уменьшении ширины экрана блок будет адаптироваться, но не превысит указанную максимальную ширину.

.container {
    max-width: 800px;
    margin: 0 auto;
    padding: 20px;
}

margin: auto по горизонтали работает только с блочными элементами, имеющими заданную ширину.

Обтекание справа
