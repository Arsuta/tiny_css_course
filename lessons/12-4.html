Урок 12.4 - Адаптивный дизайн

Что такое адаптивный дизайн?

Адаптивный дизайн - это подход к веб-дизайну, который позволяет веб-страницам адаптироваться к различным размерам экранов и устройствам.

Основные принципы адаптивного дизайна:

  • Гибкость - использование гибких сеток и изображений, которые могут изменять свои размеры.
  • Медиа-запросы - применение различных стилей в зависимости от характеристик устройства (ширина, высота, разрешение).
  • Удобство использования - создание интерфейсов, которые легко использовать на мобильных устройствах.

Медиа-запросы

Медиа-запросы позволяют применять разные стили в зависимости от характеристик устройства:

/* Пример Media Query для мобильных устройств */
@media (max-width: 600px) {
    .responsive-item {
        font-size: 14px;
    }
}

Гибкие изображения

Гибкие изображения автоматически подстраиваются под размеры контейнера:

/* Гибкие изображения */
img {
    max-width: 100%;
    height: auto;
}
