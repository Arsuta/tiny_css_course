Урок 11.4 - Стилизация блока

Структура компонента

Компонент социальных кнопок построен по методологии BEM, где каждый элемент имеет свою роль и стилизацию:

Основные элементы:

  • Блок (.social-share) - контейнер, определяющий контекст позиционирования
  • Кнопка (.social-share__button) - триггер для открытия меню
  • Меню (.social-share__menu) - выпадающий список с соцсетями
  • Ссылки (.social-share__link) - элементы для шаринга
Facebook Twitter

Принципы стилизации блока:

  • Изоляция - компонент не должен зависеть от внешних стилей
  • Модульность - возможность переиспользования в разных местах
  • Гибкость - адаптация под разные размеры и темы
  • Доступность - работа с клавиатурой и скринридерами
/* Основной блок */
.social-share {
    position: relative;
    display: inline-block;
}

/* Кнопка */
.social-share__button {
    width: 48px;
    height: 48px;
    border-radius: 50%;
    border: none;
    background: #466A85;
    color: white;
    cursor: pointer;
    transition: all 0.3s ease;
}

Позиционирование меню

Для правильного позиционирования выпадающего меню используется комбинация CSS свойств:

Ключевые свойства позиционирования:

  • position: relative на родителе

    Создает контекст позиционирования для абсолютно позиционированных дочерних элементов

  • position: absolute на меню

    Позволяет позиционировать меню относительно родителя

  • transform: translateX(-50%)

    Центрирует меню относительно кнопки

Особенности реализации:

  • z-index - управление слоями при наложении элементов
  • box-shadow - создание эффекта глубины
  • margin-top - отступ от кнопки для визуального разделения
  • border-radius - скругление углов для современного вида
/* Меню */
.social-share__menu {
    position: absolute;
    top: 100%;
    left: 50%;
    transform: translateX(-50%);
    margin-top: 10px;
    background: white;
    border-radius: 8px;
    box-shadow: 0 2px 8px rgba(0,0,0,0.2);
}

Стилизация состояний

Компонент имеет несколько состояний, которые требуют особой стилизации:

Основные состояния:

  • Hover - при наведении курсора
  • Focus - при фокусе с клавиатуры
  • Active - при нажатии
  • Open - когда меню открыто
/* Состояния */
.social-share__button:hover {
    background: #385674;
    transform: scale(1.1);
}

.social-share__button:focus {
    outline: none;
    box-shadow: 0 0 0 3px rgba(70, 106, 133, 0.5);
}

.social-share__button:active {
    transform: scale(0.95);
}
