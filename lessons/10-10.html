Урок 10.10 - Режим принудительных цветов

Что такое forced-colors?

Режим принудительных цветов (forced-colors) - это системная настройка, которая заменяет цвета сайта на определенную ограниченную палитру для улучшения читаемости.

Системные цвета в forced-colors:

  • Canvas - цвет фона
  • CanvasText - цвет текста
  • LinkText - цвет ссылок
  • Highlight - цвет выделения
  • HighlightText - цвет текста на выделении
/* Адаптация под forced-colors */
@media (forced-colors: active) {
    .form-control {
        border-color: CanvasText;
        background-color: Canvas;
        color: CanvasText;
    }

    .form-control:focus {
        outline-color: Highlight;
    }

    .form-label {
        color: CanvasText;
    }
}

Применение forced-colors

Рассмотрим как адаптировать различные элементы формы под режим принудительных цветов:

Поле ввода

Кнопка

/* Стили для разных элементов */
@media (forced-colors: active) {
    .button {
        forced-color-adjust: none;  /* Отключаем принудительные цвета */
    }

    .form-control:focus {
        outline-color: Highlight;
    }
}
