Урок 6.6: Настройка размера и позиции фона

Теория

Свойства background-size и background-position

background-size
Управляет размером фона
background-position
Управляет положением фона

Демонстрация background-size

cover
background-size: cover;    /* Заполняет контейнер */
background-size: contain;  /* Вписывается целиком */
background-size: 100%;     /* Точный размер */

Демонстрация background-position

50%, 50%
background-position: center;
background-position: top right;
background-position: 50% 50%;

Практика

Задание: Настройка фона для карточки профиля

Создайте карточку с настроенным фоновым изображением в header.

Требования:

  • Добавьте фоновое изображение в header
  • Настройте правильный размер изображения
  • Задайте оптимальное позиционирование
  • Добавьте градиент поверх изображения

Подсказки:

  • Используйте background-image для добавления изображения
  • Примените background-size: cover для заполнения
  • Задайте background-position: center для центрирования
  • Используйте linear-gradient с background-image
