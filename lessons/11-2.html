Урок 11.2 - Настройка проекта

Структура проекта

При создании компонента важно правильно организовать структуру проекта. Это поможет в дальнейшем поддерживать и масштабировать код.

Принципы организации:

  • Модульность - разделение кода на независимые модули
  • Переиспользуемость - возможность использовать компоненты в разных местах
  • Масштабируемость - легкость добавления новых функций
  • Поддерживаемость - простота внесения изменений
project/
├── index.html
├── css/
│ ├── main.css
│ ├── social-share.css
│ └── variables.css
├── js/
│ └── social-share.js
└── images/
└── icons/

Описание файлов:

  • index.html - основной HTML файл
  • main.css - основные стили
  • social-share.css - стили компонента
  • variables.css - CSS переменные
  • social-share.js - JavaScript для интерактивности

HTML структура

Компонент социальных кнопок состоит из нескольких ключевых элементов:

Элементы компонента:

  • Контейнер (.social-share) - обертка всего компонента
  • Кнопка (.social-share__button) - триггер для открытия меню
  • Меню (.social-share__menu) - выпадающий список с соцсетями
  • Ссылки (.social-share__link) - кнопки для шаринга

CSS переменные

Определим основные переменные для компонента:

/* variables.css */
:root {
    /* Цвета */
    --color-facebook: #1877f2;
    --color-twitter: #1da1f2;
    --color-linkedin: #0077b5;
    
    /* Размеры */
    --button-size: 48px;
    --icon-size: 24px;
    
    /* Анимации */
    --transition-speed: 0.3s;
}

Базовые стили

Добавим начальные стили для компонента:

/* social-share.css */
.social-share {
    position: relative;
    display: inline-block;
}

.social-share__button {
    width: var(--button-size);
    height: var(--button-size);
    border-radius: 50%;
    border: none;
    cursor: pointer;
}

.social-share__menu {
    position: absolute;
    display: none;
    padding: 8px;
    background: white;
    box-shadow: 0 2px 8px rgba(0,0,0,0.2);
}
