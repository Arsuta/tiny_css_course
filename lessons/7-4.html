Урок 7.4 - Обтекание текстом изображения собаки

1. Использование path() для сложных форм

Свойство path() позволяет создавать сложные формы обтекания, но пока имеет ограниченную поддержку браузерами. Рассмотрим альтернативные подходы:

/* Использование polygon() вместо path() */
.complex-shape {
    shape-outside: polygon(30% 0%, 70% 0%, 100% 30%, 100% 70%, 70% 100%, 30% 100%, 0% 70%, 0% 30%);
    clip-path: polygon(30% 0%, 70% 0%, 100% 30%, 100% 70%, 70% 100%, 30% 100%, 0% 70%, 0% 30%);
}

2. Настройка shape-margin

shape-margin позволяет создать отступ между текстом и формой обтекания:

Без отступа (shape-margin: 0)

С отступом (shape-margin: 20px)

/* Добавление отступа */
.image-with-margin {
    shape-margin: 20px;    /* Отступ от формы */
    margin-right: 20px;    /* Обычный отступ справа */
}

4. Ключевые моменты

  • polygon() - альтернатива path() для создания сложных форм
  • shape-margin - управление отступом между текстом и формой
  • clip-path - визуальная обрезка изображения
  • float - базовое свойство для обтекания
