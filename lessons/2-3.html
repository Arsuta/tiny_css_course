Урок 2.3: Треки и линии сетки

Теория

Треки сетки

Треки - это пространство между двумя соседними линиями сетки, которое может быть горизонтальным (ряды) или вертикальным (колонки).

Определение треков

.grid {
    display: grid;
    /* Колонки */
    grid-template-columns: 100px 1fr 2fr;
    /* Ряды */
    grid-template-rows: auto minmax(100px, auto) 200px;
}

Единицы измерения треков

  • px, em, rem - фиксированные размеры
  • % - процент от размера контейнера
  • fr - доля свободного пространства
  • auto - размер на основе содержимого
  • minmax(min, max) - адаптивный размер

Линии сетки

Нумерация линий

.item {
    /* От 1 до 3 линии */
    grid-column: 1 / 3;
    /* От 2 до 4 линии */
    grid-row: 2 / 4;
}

Именованные линии

.grid {
    grid-template-columns: 
        [sidebar-start] 200px 
        [content-start] 1fr 
        [content-end];
}

.sidebar {
    grid-column: sidebar-start / content-start;
}

Функция repeat()

Простое повторение

/* 3 колонки по 1fr */
grid-template-columns: repeat(3, 1fr);

/* 4 ряда по 100px */
grid-template-rows: repeat(4, 100px);

Сложные паттерны

/* Чередование размеров */
grid-template-columns: repeat(2, 100px 1fr);

/* Автозаполнение */
grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));

Важные моменты

  • Линии нумеруются начиная с 1
  • Можно использовать отрицательные номера линий (справа налево)
  • fr распределяет оставшееся пространство
  • minmax() помогает создавать адаптивные сетки
