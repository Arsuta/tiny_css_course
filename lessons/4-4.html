Урок 4.4: Многоколоночная верстка

Теория

CSS Multi-column Layout Module

Модуль многоколоночной верстки CSS позволяет разделить контент на несколько колонок, подобно газетному макету.

Медиа-запросы для адаптивности

Для создания адаптивного макета используются медиа-запросы, которые изменяют количество колонок в зависимости от ширины экрана.

/* Для средних экранов */
@media (max-width: 768px) {
    .column-example {
        column-count: 2;
    }
}

/* Для маленьких экранов */
@media (max-width: 480px) {
    .column-example {
        column-count: 1;
    }
}

Разделители колонок

Свойство column-rule позволяет добавить разделительную линию между колонками.

Пример текста с разделителями между колонками. Обратите внимание на вертикальные линии.

Разделители помогают визуально отделить колонки друг от друга, улучшая читаемость.

.column-example {
    column-rule: 1px solid #466A85;
    column-rule-width: 1px;
    column-rule-style: solid;
    column-rule-color: #466A85;
}

Контроль разрывов контента

CSS предоставляет свойства для контроля того, как контент разбивается между колонками.

Неразрывный блок

Этот блок не будет разбиваться между колонками.

А этот текст может быть разбит между колонками как обычно.

.break-control {
    break-inside: avoid;
    page-break-inside: avoid;
}

Элементы на всю ширину

Свойство column-span позволяет элементу занимать всю ширину, игнорируя колонки.
