Урок 2.4: Grid template areas

Теория

Grid Template Areas

Grid Template Areas позволяет создавать макеты с помощью именованных областей, делая код более наглядным и понятным.

Определение областей

.grid {
    display: grid;
    grid-template-areas:
        "header header header"
        "sidebar content content"
        "footer footer footer";
    grid-template-columns: 200px 1fr 1fr;
    grid-template-rows: auto 1fr auto;
    gap: 20px;
}

Привязка элементов к областям

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }

Пустые ячейки и точки

Использование точки для пустой ячейки

.grid {
    grid-template-areas:
        "header header ."
        "sidebar content content"
        ". footer footer";
}

Адаптивные макеты

Изменение макета с помощью медиа-запросов

@media (max-width: 768px) {
    .grid {
        grid-template-areas:
            "header"
            "content"
            "sidebar"
            "footer";
        grid-template-columns: 1fr;
    }
}

Важные правила

  • Каждая строка должна иметь одинаковое количество ячеек
  • Область должна формировать прямоугольник
  • Имена областей должны быть уникальными
  • Точка (.) обозначает пустую ячейку
