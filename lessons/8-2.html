Урок 8.2 - Оформление корзины

1. Типографика

Правильная типографика играет важную роль в создании удобного интерфейса корзины:

Заголовок корзины

Основной текст с описанием товаров и условий доставки.

Дополнительная информация мелким шрифтом. 
/* Стили для типографики */
h3 {
    font-family: Arial, sans-serif;
    color: #466A85;
    font-size: 1.5em;
    margin-bottom: 1em;
}

p {
    line-height: 1.6;
    color: #333;
}

small {
    color: #666;
    font-size: 0.875em;
}

2. Кнопки и ссылки

Различные типы кнопок для разных действий в корзине:

/* Основная кнопка */
.primary-button {
    background: #466A85;
    color: white;
}
/* Вторичная кнопка */
.secondary-button {
    background: #f7f5e8;
    color: #466A85;
    border: 1px solid;
}

3. Поля ввода

Стилизация полей ввода для форм:

/* Стили для полей ввода */
.form-input {
    padding: 10px;
    border: 1px solid #ddd;
    border-radius: 4px;
}
