Урок 9.4 - Типографика для кредитной карты

1. Подключение шрифтов

Для создания реалистичной кредитной карты важно использовать правильные шрифты. Рассмотрим основные аспекты работы со шрифтами:

Форматы шрифтов:

  • WOFF2 - Самый современный и эффективный формат, меньший размер файла
  • WOFF - Хорошая поддержка в браузерах, запасной вариант
  • TTF/OTF - Устаревшие форматы, используются как последний fallback
/* Подключение шрифта */
@font-face {
    font-family: 'CreditCard';
    src: url('fonts/credit-card.woff2') format('woff2'),
         url('fonts/credit-card.woff') format('woff');
    font-display: swap;
}

Пояснение свойств @font-face:

  • font-family - Задаёт имя шрифта для использования в CSS
  • src - Указывает путь к файлам шрифта и их форматы
  • font-display: swap - Показывает fallback-шрифт пока загружается основной
  • font-weight, font-style - Определяют начертание шрифта

2. Создание fallback с @supports

Важно обеспечить корректное отображение даже при отсутствии основного шрифта. Рассмотрим стратегии создания fallback:

Стратегии fallback:

  • Системные шрифты - Используйте похожие системные шрифты как запасной вариант
  • Множественные fallback - Указывайте несколько альтернативных шрифтов
  • Проверка поддержки - Используйте @supports для определения доступности шрифта
/* Проверка поддержки шрифтов */
@supports (font-family: 'CreditCard') {
    .card-number {
        font-family: 'CreditCard', monospace;
    }
}

.card-number {
    font-family: monospace;
}

Почему важен fallback:

  • Скорость загрузки - Пользователь видит контент, пока загружается основной шрифт
  • Совместимость - Сайт корректно отображается даже при проблемах с загрузкой шрифта
  • Доступность - Контент остаётся читаемым в любых условиях

3. Улучшение типографики

Для создания профессионального вида карты важно правильно настроить типографику. Рассмотрим основные приёмы:

Ключевые аспекты типографики:

  • Межбуквенное расстояние - letter-spacing улучшает читаемость цифр
  • Тени текста - text-shadow добавляет глубину и контраст
  • Размер шрифта - Должен быть достаточным для чтения, но не слишком большим
  • Регистр текста - text-transform помогает выделить важную информацию
/* Улучшение типографики */
.card-number {
    letter-spacing: 2px;
    text-shadow: 1px 1px 2px rgba(0,0,0,0.3);
}

.card-holder {
    text-transform: uppercase;
    font-size: 14px;
}
