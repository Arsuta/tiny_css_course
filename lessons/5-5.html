Урок 5.5: Анимация при наведении и focus-within

Теория

Анимация при наведении

Добавим интерактивность к нашим карточкам с помощью hover-эффектов:

.card {
    transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.card:hover {
    transform: translateY(-5px);
    box-shadow: 0 8px 16px rgba(0,0,0,0.2);
}

Наведите на карточку

При наведении карточка приподнимется и тень станет более заметной.

Важные моменты:

  • transition определяет, какие свойства будут анимированы
  • transform используется для плавного перемещения
  • box-shadow создает эффект поднятия карточки

Использование focus-within

Псевдокласс focus-within позволяет стилизовать контейнер, когда любой его дочерний элемент получает фокус:

.card:focus-within {
    box-shadow: 0 0 0 3px #466A85;
    outline: none;
}

Преимущества focus-within:

  • Улучшает доступность интерфейса
  • Работает с любыми интерактивными элементами
  • Позволяет стилизовать родительские элементы

Комбинирование эффектов

Давайте объединим hover-эффекты и focus-within:

.card {
    transition: all 0.3s ease;
}

.card:hover, .card:focus-within {
    transform: translateY(-5px);
    box-shadow: 0 8px 16px rgba(0,0,0,0.2);
}

Практика

Задание: Создание интерактивных карточек

Создайте набор карточек с анимацией при наведении и фокусе.

Требования:

  • Добавьте эффект поднятия карточки при наведении
  • Реализуйте подсветку карточки при фокусе на интерактивных элементах
  • Используйте плавные переходы для всех анимаций
  • Добавьте дополнительные эффекты на ваш выбор

Подсказки:

  • Используйте transform: translateY() для эффекта поднятия
  • Примените transition для плавности анимаций
  • Добавьте box-shadow для визуального выделения
  • Не забудьте про :hover и :focus-within
